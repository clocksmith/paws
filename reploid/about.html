What is REPLOID?

REPLOID is a browser-native agentic AI system that can modify its own source code through structured, human-supervised workflows. Built on the PAWS philosophy (Prepare Artifacts With SWAP), it demonstrates that the browser is the ideal environment for Recursive Self-Improvement (RSI) agents.

The Core Innovation: While most AI coding assistants operate as black boxes, REPLOID shows you beautiful interactive diffs, lets you approve files individually, and maintains a complete audit trail in git. It's AI-assisted development with the transparency and control of a senior engineer.

Why the Browser?

  • Rich Web APIs for visualization and interaction
  • Built-in sandboxing for safe code execution (Web Workers)
  • Visual feedback loops that surpass command-line tools
  • Universal deployment with zero installation required
  • Local-first with IndexedDB for complete privacy

The DREAMER Workflow

REPLOID follows a structured 8-state finite state machine:

  1. IDLE â†’ Set your goal to begin
  2. CURATING_CONTEXT â†’ Agent selects relevant files using AI
  3. AWAITING_CONTEXT_APPROVAL â†’ You review and approve the file selection
  4. PLANNING_WITH_CONTEXT â†’ Agent analyzes code and plans changes
  5. GENERATING_PROPOSAL â†’ Creates explicit change bundle (dogs.md)
  6. AWAITING_PROPOSAL_APPROVAL â†’ Interactive diff viewer for change review
  7. APPLYING_CHANGES â†’ Applies approved changes with git checkpoint
  8. REFLECTING â†’ Learns from outcome for continuous improvement

Human-in-the-Loop: You have full control at stages 3 and 6. Nothing happens without your explicit approval. This isn't autonomous coding - it's augmented engineering.

Architecture Highlights

  • Modular Design - 40+ independently testable modules with dependency injection
  • Event-Driven - Loose coupling via event bus pattern for extensibility
  • Git-Based VFS - Virtual filesystem with full checkpoint/rollback capabilities
  • Web Worker Sandboxing - Safe code execution and verification in isolation
  • IndexedDB Storage - Client-side persistence for sessions and reflections
  • 85 Passing Tests - Comprehensive unit and integration test coverage
  • Zero Build Step - ES modules work directly in modern browsers

Security & Privacy

Local-First Architecture: Everything runs in your browser. Your code, your data, your API keys - all stored locally in IndexedDB. REPLOID never sends your code anywhere except to the LLM API you explicitly configure.

  • Browser Sandboxing - Origin isolation and Web Worker execution
  • No Telemetry - Zero tracking, zero analytics, zero data collection
  • Git Audit Trail - Every change tracked with full commit history
  • Selective Approval - Review and approve changes file-by-file
  • Checkpoint Rollback - Instant revert to any previous state
